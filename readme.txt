METAL GLOVE SOLID*
--------------------

A Gauntlet clone.
by Stephen Carlyle-Smith (stephen.carlylesmith@googlemail.com)


* This game is very early beta!


Controls:
--------
Arrow keys to move,
Ctrl to shoot
S - Show spellbook
1..9 - Cast spell
P - Pause
